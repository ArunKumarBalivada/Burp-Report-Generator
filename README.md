# Burp Report Generator

A Burp Suite extension for automated penetration testing report generation.

## Features

- Project Management
- Finding Library
- Finding Management
- JSON Project Save/Open
- DOCX Report Generation
- Executive Summary
- Vulnerability Summary
- Detailed Findings
- Dashboard
- Settings

## Technologies

- Java 21
- Maven
- Apache POI
- Jackson
- Burp Montoya API
- Swing
